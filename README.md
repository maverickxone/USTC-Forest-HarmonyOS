# USTC-Forest - HarmonyOS NEXT App

**USTC-Forest** 是一个为中科大学生打造的专注森林应用，旨在帮助同学们提高约束力，通过专注计时、种树养成甚至强制锁机的方式，激励大家高效学习和工作。该应用基于 **HarmonyOS NEXT** 开发，计划迭代多个版本。

目前开发的 **Mark 1** 版本预计提供专注计时、树木生长、枯萎机制、森林展示、限制应用使用等基础功能。

---

## 📱 基础功能演示

- **专注计时**：每次专注的时长由用户设定（5分钟-180分钟），点击开始专注后种下一颗种子，种子的成长形态根据时间会有变化，做到时间可视化。
- **树木生长**：专注完成后，会根据专注时长生成一棵树，提供正反馈，激励用户再接再厉。
- **森林展示**：展示已种植的树木，每棵树代表一次专注。用户可以按日、周、月、年展开查看自己的专注历史。
- **枯萎机制**：中途放弃的专注会让树苗枯萎，代表一次未完成的任务。
- **锁定应用**：通过用户手动打开权限，可支持黑名单/白名单两种形态，用于强制性专注或排除社交媒体的干扰。

## 🔧 技术栈

- **语言**：ArkTS（TypeScript 超集）
- **UI 框架**：ArkUI（声明式 UI）
- **开发平台**：DevEco Studio
- **适配版本**：HarmonyOS NEXT

---

## ⚙️ 开发进度

- **Mark 1**：目前开发的 demo 版本，支持上述提及的基础功能。
- **Mark 2**：进一步适配鸿蒙原生态，如支持小窗，实况窗等。
- **Mark 3**：全面完善，支持校内排行，社交互动等后期非必须功能。

### 一些可能的后续设想（未成形）：

- **轻量简约版**：Mark 5，页面简约，不支持排行榜等。
- **稳定增强版**：Mark 44，根据反馈进行优化。
- **官方协作版本**：Mark 85
